# Sistema-de-estoques


Sistema de Estoque Final

Este projeto foi desenvolvido com o objetivo de auxiliar no controle e organização de produtos em estoque, permitindo uma administração mais prática dos itens cadastrados.

O sistema encontra-se funcional pelo Codespaces e, durante o desenvolvimento, foram realizadas melhorias e ajustes em algumas partes do projeto, incluindo alterações no HTML para implementação de funcionalidades, como o botão de adicionar produtos ao estoque.

Como rodar o projeto

Abra o terminal e entre na pasta do projeto:
cd karla

Instale as bibliotecas necessárias:
pip install -r requirements.txt

A saída do terminal deve estar em:

/workspaces/Sistema-de-estoques/karla

Execute o servidor com o comando:
python manage.py runserver 0.0.0.0:8000

Após isso, o sistema deverá abrir normalmente.

Área Administrativa

Para acessar a área administrativa do estoque, adicione /admin ao final do link gerado pelo Codespaces.

Caso seja necessário criar um administrador, utilize o comando:

python manage.py createsuperuser

No campo de e-mail, basta apertar Enter para ignorar.
